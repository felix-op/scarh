# Módulo 4: Análisis de los Datos

Este módulo provee las herramientas necesarias para visualizar, analizar y exportar la información histórica registrada.

### RF [4.01] Historial completo de datos
* **Usuarios**: Administrador, usuarios.
* **Descripción**: Los usuarios del sistema tienen acceso a todos los datos que llegaron al sistema, con la información de la fecha y a que limnígrafo corresponde.

### RF [4.02] Comparación de datos
* **Usuario**: Administrador, usuarios.
* **Descripción**: Podrá ver paralelamente los datos de varios limnígrafos y en distintos períodos de tiempo. Como una tabla o lista.

### RF [4.03] Gráficos interactivos
* **Usuario**: Administrador, usuarios.
* **Descripción**: Los usuarios ven la evolución de los datos en distintos tipos de gráficos (líneas, comparaciones, tendencias, etc.).

### RF [4.04] Filtros y búsquedas
* **Usuarios**: Administrador, usuarios.
* **Descripción**: El sistema debe permitir a los usuarios aplicar filtros por fecha, palabras clave y otros criterios relevantes para facilitar la localización y análisis de datos. Por ejemplo:
  * Historial de datos (consulta general de registros).
  * Generación de gráficos (para ajustar los datos visualizados).
  * Exportación de datos (para descargar solo la información filtrada en CSV, Excel o PDF).

### RF [4.05] Descarga de información
* **Usuarios**: Administrador, usuarios.
* **Descripción**: Los datos y reportes podrán exportarse en formatos como CSV, Excel o PDF.
