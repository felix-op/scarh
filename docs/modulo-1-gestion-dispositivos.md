# Módulo 1: Gestión de los Dispositivos

Este módulo abarca la administración, configuración y ubicación de los limnígrafos en el sistema.

### RF [1.01] Gestión de limnígrafos
* **Usuarios**: Administrador, Técnico.
* **Descripción**: El sistema permite dar de alta, editar y dar de baja el registro de los dispositivos limnígrafos en la plataforma.

### RF [1.02] Ubicación geográfica
* **Usuarios**: Administrador, Estadístico.
* **Descripción**: El sistema permite guardar las coordenadas (X, Y) de cada limnígrafo para identificarlo en un mapa.

### RF [1.03] Gestión de rutas de acceso a los limnígrafos
* **Usuarios**: Administrador y Técnico.
* **Descripción**: El sistema permitirá asociar a cada limnígrafo una ruta de acceso mediante la carga de un archivo GPX o KML generado por una aplicación externa. La ruta podrá visualizarse en el mapa y descargarse para utilizarla en una aplicación móvil de navegación. El sistema no proporcionará navegación GPS en tiempo real.

### RF [1.04] Estados de los dispositivos
* **Usuarios**: Administrador, Técnico.
* **Descripción**: Marcar si un equipo está activo, advertencia, peligro o fuera de rango.

### RF [1.05] Mapa de color
* **Usuarios**: Administrador, usuarios.
* **Descripción**: Mostrar en el mapa con color los sectores del río que abarque el limnígrafo a decisión del usuario.
